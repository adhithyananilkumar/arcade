# Naming Conventions

Documented naming standards for consistency across the codebase.

- **Components**: `PascalCase` (e.g., `UserProfile.tsx`)
- **Hooks**: `useSomething` (e.g., `useAuth.ts`)
- **Variables**: `camelCase` (e.g., `userList`)
- **Constants**: `UPPER_SNAKE_CASE` (e.g., `MAX_RETRIES`)
- **Files**: `kebab-case` (e.g., `user-service.ts`) except React components which follow `PascalCase`.
